# Client UI (Mahalanobis RecSys)

Interfaz de usuario orientada a estudiantes de colegio para el sistema de recomendación de carreras.

## Enfoque de Diseño (Premium & Dynamic)
La interfaz debe impresionar desde el primer momento, sintiéndose como un producto de alta calidad y no solo una "calculadora académica".

- **UI/UX**: Estética moderna, vibrante, uso estratégico de espacio y tipografía de alta calidad.
- **Flujo de Usuario**: Formulario dinámico y rápido de baja fricción para ingresar notas e intereses.
- **Explicabilidad Visual**: La pantalla final incluye un Gráfico de Radar (Telaraña) interactivo que superpone el perfil real del estudiante con el perfil "ideal" (centroide) de la carrera recomendada.

## Flujo de Pantallas
1. **Landing/Formulario**: Breve introducción y formulario de 5-6 áreas clave.
2. **Loading (Opcional)**: Pequeña animación indicando que la IA "Calcula distancias matemáticas".
3. **Veredicto**: Presentación visual del Top 3 de carreras, con el Radar Chart como elemento central.
